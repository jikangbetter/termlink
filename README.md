# TermLink

A cross-platform terminal emulator with SSH and SFTP support, built with Rust and egui.

## Installation

### From Source

```bash
# Clone the repository
git clone https://github.com/jikangbetter/termlink.git
cd termlink

# Build the project
cargo build --release

# Run the application
cargo run --release
```

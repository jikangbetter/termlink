# TermLink

一个基于 Rust 和 egui 构建的跨平台终端模拟器，支持 SSH 和 SFTP。

## 安装

### 从源码编译

```bash
# 克隆仓库
git clone https://github.com/jikangbetter/termlink.git
cd termlink

# 构建项目
cargo build --release

# 运行程序
cargo run --release
```
